Executive Summary

Cursor je pokročilý editor kódu (fork Visual Studio Code) s integrovanými AI agenty, který pracuje s lokálními soubory, píše skripty a má plný přístup ke složkové struktuře. Klíčová výhoda oproti čistému chatu v prohlížeči: Cursor analyzuje soubory v projektu, napíše skript a úkol provede programově — matematicky správně, bez nutnosti umět programovat. V aktuálních stable buildech (např. 2.6.x) přibývá mimo jiné lepší podpora MCP včetně interaktivních výstupů a správy pluginů pro týmy.

🧠

4 módy práce

Agent, Plan, Debug, Ask — od přímého vykonání po bezpečný průzkum.

Verze a novinky (Cursor 2.6)

📌 Referenční build (kontrola v Help → About)

Průvodce je srovnán s aktuálním stable instalátorem na Windows. Typické hodnoty:

  • Cursor: 2.6.20 (system setup)
  • Základ VS Code: 1.105.1
  • Release track: Default · Build: Stable
  • Electron: 39.8.1 · Chromium: 142.x · Node.js: 22.22.x
  • OS: Windows 10/11 x64 (hodnota win32 v About)

Přesné číslo sestavení a commit ID vždy ověřte v editoru — menší patchové verze (2.6.20, 2.6.21, …) se v průběhu týdnů mění.

✨ Novinky v řadě 2.6 (březen 2026)

Podle oficiálního changelogu 2.6 (3. 3. 2026):

  • MCP Apps — interaktivní rozhraní přímo v agentím chatu: např. grafy (Amplitude), diagramy (Figma), tabule (tldraw). Dokumentace: MCP Apps.
  • Team marketplaces pro pluginy — u plánů Teams a Enterprise mohou administrátoři sdílet interní pluginy s řízením přístupu (team marketplaces).
  • Vylepšení jádra — řada desktopových úprav a oprav; posílení mimo jiné Debug módu.
🔗
Kompletní přehled vydání: cursor.com/changelog. U modelů sledujte také oznámení k Composer / Composer 2 v changelogu — výběr modelu je vždy v rozhraní chatu.

Rozhraní a nastavení

🎨 Personalizace vzhledu

  • Settings → Theme: světlé/tmavé téma, fonty — komunitní témata ke stažení
  • View → Zoom: zvětšení rozhraní pro lepší čitelnost
  • Přepínače vpravo nahoře: stromová struktura, agenti (chat), obsah souborů
  • Pozice panelů je individuální — strom vlevo nebo vpravo dle preference
💡
Při prvním otevření zvolte světlé téma — snižuje počáteční stres z nového prostředí. Zvětšete si Zoom pro příjemnější práci.
📂

Workspace — výběr složky

Cursor potřebuje vědět, s čím má pracovat. Při Open vybíráte složku — to je váš workspace.

📁

Přístup 1: Jedna složka

Konkrétní složka jednoho projektu. Ideální pro zaměřenou práci na jednom úkolu.

🌳

Přístup 2: Celý strom

Celá stromová struktura (např. pro Vibe Coding). Cursor vidí více projektů najednou.

ℹ️
Cursor nemá přístup mimo vybranou složku. Použijte New Window pro další projekt vedle, nebo Open pro změnu složky.

📎 Odkazování na soubory

  • Přetáhnutí složky/souboru přímo do chatu
  • Zavináč @ + název souboru
  • Pravé tlačítko → Copy Path — vložení cesty do chatu
🎛️

Módy a modely

▶️

Agent

Automaticky vykoná, co řeknete. Přímá akce bez plánování.

📝

Plan

Analyzuje → vytvoří plán → pak provádí. Pro složité úkoly (80 % Tomovy práce).

🐛

Debug

Hledání a oprava chyb v kódu. Specializovaný mód pro diagnostiku; v řadě 2.6 proběhla řada vylepšení jádra tohoto módu.

❓

Ask

Nic nemění, pouze odpovídá. Bezpečný pro průzkum a dotazy o souborech.

💡
Začínejte s Ask pro průzkum (nic nepokazíte), Plan pro složité úkoly s mnoha kroky, Agent pro jednoduché přímé akce.
🏆

Opus

Pro složité úkoly a velké množství dat. Nejschopnější, ale dražší.

⚡

Sonnet

Levnější, podobně inteligentní. Ideální pro běžné a jednodušší úkoly.

🚀

Composer

Vlastní model Cursoru — extrémně rychlý, vhodný na iteraci; v roce 2026 changelog zmiňuje i Composer 2 — dostupnost a přesný název vždy v rozhraní výběru modelu.

🔄

Auto

Cursor si vybere sám. Méně spolehlivé — nedoporučeno pro důležité úkoly.

⚠️
Doporučení: Anthropic modely (Opus, Sonnet) místo Auto. Vybírejte model explicitně dle složitosti úkolu.

💰 Náklady

Předplatné ~200 USD ≈ 150 USD na tokeny; až 3000 USD při výběru dražších modelů. Šetřete: jednoduché úkoly v Auto, složité na Opus/Sonnet.

Kontext — jak Cursor „vidí"

🔑 Zlaté pravidlo

Čím konkrétnější kontext, tím lepší výsledek. Bez odkazu Agent prohledá celý workspace — u tisíců souborů pomalé a nepřesné.

Přetáhněte složku do chatu

Zužuje kontext — Cursor ví, kde hledat.

Zavináč @ + soubor

Odkaz na konkrétní soubor pro maximální přesnost.

Copy Path

Pravé tlačítko → Copy Path — vložení přesné cesty do chatu.

Inbox složka

Složka na „bordel" — Cursor si pamatuje kontext z předchozích dotazů.

💬
„Kontext je nová ropa" — bez něj nelze vytvořit praktický výstup. Vždy směrujte Cursor na konkrétní data.

Rules, Skills, Commands

📐 Rules — chování agenta

Ovlivňují jak se agent chová — jako osobnostní nastavení (tón, pravidla, konvence).

  • Globální: Cursor Settings → Rules (uživatelská pravidla pro všechny projekty)
  • Projektová: složka .cursor/rules v workspace (nebo pravidla v nastavení projektu dle dokumentace)
  • Custom příkazy (Commands): můžete mít v .cursor/commands — to jsou uložené prompty, ne totéž co Rules
  • Příklad: „Když řeknu Aibility jazyk, použij tento tón voice"
  • Příklad: „Když říkám design systém, je to tento soubor"
  • Instalace z Kila (nákup Rules packů)

🛠️ Skills — schopnosti agenta

Rozšiřují co agent umí — jako naučit ho nový postup.

  • Příklad: Skill „Implement Design" — jak pracovat s Figmou
  • Cursor může vytvořit Skill z konverzace: „Vytvoř mi z toho Skill"
  • Opakující se úkoly → uložit jako Skill

⌨️ Commands — uložené prompty

Předpřipravené/uložené prompty na lomítko — rychlé spuštění opakovaných úkolů.

  • /md — zformátuje text do strukturovaného Markdownu
  • /share — nahraje MD na Vercel, vrátí URL pro sdílení
  • Command může obsahovat i skript k zavolání
🔌

MCP servery a externí nástroje

MCP (Model Context Protocol) = konfigurace pro připojení k externím aplikacím. Od verze 2.6 navíc MCP Apps — servery mohou do chatu vykreslovat interaktivní UI (grafy, diagramy, tabule), ne jen posílat text.

  • Slack — připojení přes autorizaci; procházení kanálů, stahování scénářů
  • Figma — designérský nástroj s integrací do vibe coding; v 2.6 i MCP App pro diagramy v chatu
  • Make.com — vizuální tvorba workflow mezi aplikacemi
  • Google Calendar — plánování přímo z Cursoru
⚠️
API vs. MCP: API je často spolehlivější (např. Asana MCP padalo). Kontrolujte „MCP" v popisu pluginu na Marketplace.
🛡️

Oprávnění a sandbox

Cursor Settings → Agents

  • Ask me — agent se ptá na povolení před každou akcí
  • Sandbox — omezené prostředí, bezpečnější
  • All (Allow) — agent dělá vše včetně MCP; rychlejší, ale vyšší riziko
🔐
Pozor na citlivá data. Lze zakázat browser a MCP dle potřeby. Každou akci v chatu lze vrátit zpět (undo).

Scénáře použití

📊 Převod a transformace dat

  • Spojení Excelů: „Mám 3 soubory, spoj je, přidej sloupec region, spočítej tržby" — Cursor napíše Python skript, nainstaluje pandas; výstup matematicky správný
  • HTML → Markdown → PDF: jeden vstup → více výstupů; skripty pro batch (70+ reportů)
  • Formátování: Command /md zformátuje nečitelný text
  • Sdílení: Command /share nahraje na Vercel, vrátí URL
  • Chybějící knihovny: nechte Cursor nainstalovat — neřešte ručně

✍️ Analýza a tvorba obsahu

  • Brief pro grafika: odkaz na metodiku, design systém, archetypy → brief za 2 minuty
  • Ideace certifikátů: 15 variant v jednom souboru, různé layouty
  • LinkedIn asistent: screenshoty postů → Rule pro váš voice → příspěvky
  • Článek z výzkumu: výzkum + feedback + přepis → hotový článek
  • Nabídka z kólu: přepis hovoru + popis služeb → nabídka za 5 minut (místo 4 hodin)

💻 Vibe coding — mini-aplikace

  • Landing Page Builder: design systém + /site → HTML stránka
  • Prezentace: HTML místo PowerPointu; úpravy z připomínek zákazníka
  • Abra reporting: API Abry → interní reporting na míru
  • Inspirační grid: složka reklam → grid s náhledy, hosting na Railway

📁 Organizace a automatizace

  • Slack + Cursor: MCP → projít kanál, stáhnout a analyzovat scénáře
  • GitHub: úložiště jako Google Drive, ale verzované
  • Priority Manager: Skill pro priority → Focus bloky do Google Calendar
  • Podcasty z reportů: report → skript dialogu → TTS → 70+ podcastů
  • Audiopohádky: prompt → pohádka → validace → obrázek → audio → web
